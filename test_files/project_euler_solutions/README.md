# Project Euler Solutions

This directory contains solutions to the first 5 Project Euler problems:

- [Problem 1](problem_1/solution.py): Multiples of 3 and 5 below 1000
- [Problem 2](problem_2/solution.py): Even Fibonacci numbers below 4,000,000
- [Problem 3](problem_3/solution.py): Largest prime factor of 600851475143
- [Problem 4](problem_4/solution.py): Largest palindrome product of two 3-digit numbers
- [Problem 5](problem_5/solution.py): Smallest multiple of 2520

All solutions have been verified and are ready for use.
